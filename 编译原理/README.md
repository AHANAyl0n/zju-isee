## 实验一、利用LEX计算文本文件的字符数等
- 实验目的：了解LEX的基本编程方法。
- 实验要求：编写一个LEX输入文件，使之生成可计算文本文件的字符、单词和行数且能报告这些数字的程序。单词为不带标点或空格的字母和/数字的序列。标点和空白格不计算为单词。

### 感受
基本上是用正则表达式,感觉还行吧,不是很难


## 实验四、利用LEX和YACC简易计算器
- 实验目的：了解YACC属性处理的基本方法。
- 实验要求：生成如下文法表示的表达式对应的计算器，要求能连续处理若干个数学表达式，直到输入结束或文件结束。
